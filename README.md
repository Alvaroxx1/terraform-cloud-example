# terraform-cloud-example
This repository is used to explore Terraform Cloud's VCS integration

- The variables are created within the Terraform Cloud
- The push automatically triggers an apply in Terraform Cloud, you can also modify this configuration to add auto-approve or more.

### Merge - Speculative Plan - Apply

- Merge pull request → You can click on View details to see the Speculative Plan → After choosing confirm mergue request you can see an apply plan executed on Terraform.
